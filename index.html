Text Replacer

Replace only words

If you select this option, the string will only be replaced if it matches completely, In other words that it is not linked to other strings.
Ex: If you want to replace "ana" by "e" and you activate this option and in the text area the words "ana" and "anabolic", only the one that makes a complete match, that is, "ana" will be replaced by "e" and "anabolic" would not undergo changes
